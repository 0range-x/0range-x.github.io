TokenзӘғеҸ–йӮЈдәӣдәӢ

Posted on 2021-09-30 In еҶ…зҪ‘ Views: Valine: 2.2k 2 mins.

Windowsдёӯзҡ„tokenеҸҜд»Ҙз”ЁжқҘеҒҡд»Җд№Ҳе‘ўпјҹ

Tokenз®Җд»Ӣ

Token жҳҜжҸҸиҝ°иҝӣзЁӢжҲ–иҖ…зәҝзЁӢдёҠдёӢж–Үе®үе…Ёзҡ„дёҖдёӘеҜ№иұЎгҖӮе®үе…ЁдёҠдёӢж–ҮеҢ…еҗ«дәҶдёҺиҜҘиҝӣзЁӢжҲ–зәҝзЁӢзҡ„иҙҰжҲ·гҖҒз»„гҖҒзү№жқғжңүе…ізҡ„жҸҸиҝ°дҝЎжҒҜгҖӮTokenиҝҳеҢ…еҗ«дёҖдәӣе…¶д»–дҝЎжҒҜпјҢеҰӮдјҡиҜқidгҖҒе®Ңж•ҙжҖ§зә§еҲ«д»ҘеҸҠUACиҷҡжӢҹеҢ–зҠ¶жҖҒгҖӮ

дёҚеҗҢзҡ„вҪӨжҲ·зҷ»еҪ•и®Ўз®—жңәеҗҺпјҢ йғҪдјҡвҪЈжҲҗвјҖдёӘ Access Token пјҢиҝҷдёӘ Token еңЁвҪӨжҲ·еҲӣе»әиҝӣзЁӢжҲ–иҖ…зәҝзЁӢж—¶дјҡиў«дҪҝвҪӨпјҢдёҚж–ӯзҡ„жӢ·иҙқпјҢиҝҷд№ҹе°ұи§ЈйҮҠдәҶAвҪӨжҲ·еҲӣе»әвјҖдёӘиҝӣзЁӢиҖҢиҜҘиҝӣзЁӢжІЎжңүBвҪӨжҲ·зҡ„жқғйҷҗгҖӮдёҖиҲ¬з”ЁжҲ·еҸҢеҮ»дёҖдёӘиҝӣзЁӢйғҪдјҡжӢ·иҙқexplorer.exe зҡ„ Access Token

WindowsжңүдёӨз§Қзұ»еһӢзҡ„Tokenпјҡ

  • Delegation token(жҺҲжқғд»ӨзүҢ): з”ЁдәҺдәӨдә’дјҡиҜқзҷ»еҪ•(дҫӢеҰӮжң¬ең°з”ЁжҲ·зӣҙжҺҘзҷ»еҪ•гҖҒиҝңзЁӢжЎҢйқўзҷ»еҪ•)
  • Impersonation token(жЁЎжӢҹд»ӨзүҢ): з”ЁдәҺйқһдәӨдә’зҷ»еҪ•(еҲ©з”Ёnet useи®ҝй—®е…ұдә«ж–Үд»¶еӨ№)

жіЁпјҡ

дёӨз§ҚtokenеҸӘеңЁзі»з»ҹйҮҚеҗҜеҗҺжё…йҷӨпјӣ

е…·жңүDelegation token(жҺҲжқғд»ӨзүҢ)зҡ„з”ЁжҲ·еңЁжіЁй”ҖеҗҺпјҢиҜҘTokenе°ҶеҸҳжҲҗImpersonation token(жЁЎжӢҹд»ӨзүҢ)пјҢдҫқж—§жңүж•Ҳ

WindowsжңүдёӨз§Қзұ»еһӢзҡ„Tokenпјҡ

  • Delegation token(жҺҲжқғд»ӨзүҢ):з”ЁдәҺдәӨдә’дјҡиҜқзҷ»еҪ•(дҫӢеҰӮжң¬ең°з”ЁжҲ·зӣҙжҺҘзҷ»еҪ•гҖҒиҝңзЁӢжЎҢйқўзҷ»еҪ•)
  • Impersonation token(жЁЎжӢҹд»ӨзүҢ):з”ЁдәҺйқһдәӨдә’зҷ»еҪ•(еҲ©з”Ёnet useи®ҝй—®е…ұдә«ж–Үд»¶еӨ№)

жіЁпјҡ

дёӨз§ҚtokenеҸӘеңЁзі»з»ҹйҮҚеҗҜеҗҺжё…йҷӨ

е…·жңүDelegation tokenзҡ„з”ЁжҲ·еңЁжіЁй”ҖеҗҺпјҢиҜҘTokenе°ҶеҸҳжҲҗImpersonation tokenпјҢдҫқж—§жңүж•Ҳ

еҸҜд»Ҙиҝҷж ·зҗҶи§ЈпјҡеҪ“еүҚзі»з»ҹдёӯзҡ„жҹҗдёӘиҝӣзЁӢжҲ–иҖ…зәҝзЁӢиғҪи®ҝй—®д»Җд№Ҳж ·зҡ„зі»з»ҹиө„жәҗпјҢеҸ–еҶідәҺеҪ“еүҚд»ӨзүҢзҡ„жқғйҷҗ

жҲ‘д»¬йҖҡиҝҮexpжҸҗжқғжҲ–иҖ…ж°ёжҒ’д№Ӣи“қзӯүеҫ—еҲ°зҡ„жқғйҷҗеҚідёәSystemпјҢеҒҮеҰӮжҲ‘д»¬еҲ©вҪӨmimikatzе’ҢhashdumpдёҚиғҪиҺ·еҫ— administratorвҪӨжҲ·зҡ„еҜҶз ҒпјҢйӮЈжҲ‘д»¬еҸӘиғҪйҖҡиҝҮд»ӨзүҢзӘғеҸ–иҝӣвҫҸйҷҚжқғпјҢиҺ·еҫ—administratorвҪӨжҲ·зҡ„shellпјҢ д»ҺвҪҪд»Ҙ administratorвҪӨжҲ·зҡ„иә«д»ҪеҗҜеҠЁжҹҗдәӣжңҚеҠЎ ( жҹҗдәӣжңҚеҠЎеҸӘиғҪйҖҡиҝҮadministratorвҪӨжҲ·еҗҜеҠЁ )

0x01 MetaspolitеңЁtoken зӘғеҸ–дёӯзҡ„еә”з”Ё

жӢҝеҲ°дәҶsystemжқғйҷҗ

дҪҝз”ЁincognitoжЁЎеқ—

еҪ“еүҚжҳҜsystemжқғйҷҗпјҢеҸҜд»ҘжҹҘзңӢеҲ°жүҖжңүзҡ„з”ЁжҲ·token

list_tokens -u

еҸҜд»ҘзӘғеҸ–еҹҹз®Ўзҡ„token

impersonate_token 'SUN\Administrator'

иҝҷдёӘж—¶еҖҷжқғйҷҗе°ұе·Із»ҸжҳҜеҹҹз®Ўзҡ„жқғйҷҗпјҢдёҺеҹҹжҺ§е»әз«ӢIPCиҝһжҺҘдёҚйңҖиҰҒеҜҶз ҒпјҲеүҚжҸҗжҳҜеҸҜд»ҘIPCиҝһжҺҘпјүпјҢ

д№ӢеҗҺеҸҜд»ҘйҖҡиҝҮwmiжү§иЎҢе‘Ҫд»ӨпјҢжҲ–иҖ…и®ЎеҲ’д»»еҠЎжү§иЎҢй©¬еӯҗ

иҝ”еӣһд№ӢеүҚзҡ„з”ЁжҲ·пјҢеҸҜд»ҘдҪҝз”Ёе‘Ҫд»Өпјҡ

rev2self

0x02 Windowsе№іеҸ°дёӢincognitoзҡ„еә”з”Ё

дёӢиҪҪең°еқҖ: https://labs.mwrinfosecurity.com/assets/BlogFiles/incognito2.zip

еҸӮиҖғжүӢеҶҢ: http://labs.mwrinfosecurity.com/assets/142/mwri_security-implications-of-windows-access-tokens_2008-04-14.pdf

еёёз”Ёе‘Ҫд»Өпјҡ

еҲ—дёҫtokenпјҡincognito.exe list_tokens -u

еҸҜд»ҘзңӢеҲ°еӣ дёәдёҚжҳҜsystemжқғйҷҗпјҢеҲ—дёҫеҮәзҡ„tokenеҫҲеұҖйҷҗпјҢеҲҮжҚўеҲ°systemжқғйҷҗзңӢдёӢ

зӘғеҸ–tokenпјҡ incognito.exe execute -c "SUN\administrator" cmd.exe

вҖӢ incognito.exe execute [options] <token> <command>

еҗҢж ·жҳҜеҹҹз®Ўзҡ„жқғйҷҗ

дёәд»Җд№ҲжіЁе…ҘеҲ°еҹҹз®Ўзҡ„иҝӣзЁӢдёӯе°ұжӢҘжңүдәҶеҹҹз®Ўзҡ„жқғйҷҗе‘ўпјҹ

иҝҷйҮҢеҸҲж¶үеҸҠеҲ°иҝӣзЁӢзҡ„е®Ңж•ҙжҖ§зә§еҲ«гҖӮ
